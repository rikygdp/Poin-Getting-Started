                      createIntegratePoin({
                          user_id: "685378368911",
                          target_id: "poinIframe",
                          client_name: "your registered name",
                          client_domain: "https://yourdomain.com"
                      });
                    </script>

API Options

Codes Type  
user_id String YES Integrated POIN SYSTEM ID for each user,
example '19318235911', and it's required.
To get this ID you should registered as our partner.
target_id String YES Your created element id,
example <div id='YourIds'></div>,
set the target_id as 'YourIds'.
client_name String YES Your registered company name with us and it's required.
client_domain String YES Your website domain that used our SDK and it's required.
example 'https://yourdomain.com'.
width String OPTIONAL Will set the iframe width, the default value is '100%' if not set.
height String OPTIONAL Will set the iframe height, the default value is '100vh' if not set.
header_height Integer OPTIONAL If you have the header to show, you can set this with the same height of your header.
